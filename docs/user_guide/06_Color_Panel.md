# 06 颜色栏（颜色/色线/最近颜色）

颜色栏用于管理当前绘画颜色，并提供“色线（描边）预设”和最近使用颜色的快速切换。

## 当前颜色（Color Indicator）

当前颜色按钮通常显示为一个小色块（不同工作区布局位置略有不同）。它代表“你现在画画/填色会使用的颜色”。

你可以：

- 单击色块：打开“颜色调整”对话框，进行更精细的取色/输入。
- 悬停查看：通常会显示当前颜色的十六进制值（例如 `#RRGGBB`）。

### 颜色调整对话框（多种取色方式）

对话框顶部可以切换多种取色模式（名称以界面显示为准），常见包括：

- HSV 方块：用方块选饱和度/明度，用色带选色相。
- 色相环：用色环选色相，内部区域调节明度/饱和度。
- RGB/HSV 数值滑杆：适合精确输入或微调。
- 面板式取色：与主界面取色面板类似的操作方式。

调整完成后点击“确认”应用；点击“取消”不更改。

## 色线预设（Color Line）

颜色栏会提供固定的“色线预设”色块（目前为：红、蓝、绿），常用于：

- 快速切换线稿/描边用色；
- 配合油漆桶“吞线”功能，让填色更贴边；
- 作为文字描边颜色的快捷入口（文字工具开启描边时）。

操作方式：

- 单击某个色线预设：会将其设为“色线颜色”，同时也会同步为当前颜色，便于立刻使用。

## 最近颜色（Recent Colors）

最近颜色用于快速回到刚用过的颜色。

常见表现：

- 面板标题栏右侧：会显示最多 5 个“最近颜色”小色块。
- 面板内容区域：当最近颜色超过 5 个时，会在正文中显示更多色块。

操作方式：

- 单击任意最近颜色色块：立即切换当前颜色。

记录规则（简要）：

- 当你在取色面板拖动并松开，或在颜色对话框中确认新颜色后，该颜色会加入“最近颜色”列表。

## 颜色选择区（色相/饱和度/明度）

颜色面板的主体通常由两部分组成：

- 颜色方块：横向控制饱和度，纵向控制明度（拖动选择）。
- 色相滑条：控制色相（Hue）。

使用建议：

1. 先在色相滑条选择大致色相。
2. 再在颜色方块里选饱和度与明度。
3. 选到满意颜色后松开鼠标/笔，颜色会被记入最近颜色。
