# 01 界面总览

本文用“栏位/面板”来描述界面结构，便于你快速定位功能入口。

## 主要栏位

1. 菜单栏：屏幕顶部，按功能分类提供命令入口（文件、编辑、图像、图层、选择、滤镜、工具、视图、工作区等）。
2. 标签栏（项目标签）：菜单栏下方，用于切换/新建/关闭项目（一个项目对应一个画布工作区）。
3. 画布工具栏：绘画/选择/变换等核心工具入口（通常位于画布左侧）。
4. 工具设置栏：当前工具的参数与开关（会随所选工具变化）。
5. 颜色栏：当前颜色、色线预设、最近使用颜色，以及颜色选择区域。
6. 图层栏：图层列表、可见/锁定/剪贴、透明度、混合模式，以及图层管理操作。

## 两种工作区布局（外观不同，功能一致）

在“工作区（Workspace）”菜单中可切换两种布局：

- Default / Floating：工具栏在左侧，右侧是合并的颜色+图层面板，颜色区与图层区之间可拖动分隔条调整高度。
- SAI2：图层面板与工具面板合并为一个大面板，内部支持横向/纵向拖拽分割，颜色区也支持高度调节。
