# 07 图层栏（图层列表与控制）

图层栏用于管理图层的显示顺序与合成方式，并提供常用的图层操作入口。

## 图层控制区（不透明度/锁定/剪贴/混合模式）

图层栏顶部通常会有一组“针对当前激活图层”的控制项（布局风格不同，排布会略有差异）。

### 撤销 / 恢复（历史）

用于回退或重做操作：

- 撤销：`Ctrl/⌘+Z`
- 恢复：`Ctrl/⌘+Shift+Z`

### 不透明度（Opacity）

用于控制当前图层的透明度（`0% ~ 100%`）。

- 数值越低：图层越透明。
- 数值越高：图层越不透明。

### 锁定（Lock Layer）

锁定后，该图层将被保护，常见影响：

- 禁止绘画/填色/橡皮擦等像素修改。
- 禁止删除、剪贴等部分操作（按钮会变灰不可点）。

### 剪贴蒙版（Clipping Mask）

将当前图层设为“剪贴蒙版”：

- 开启后：当前图层内容会被下方一层（作为“底层”）的可见范围所限制。
- 常用于：在不破坏底层线稿/形状的前提下上色或加阴影。

提示：

- 剪贴图层通常会在列表中有缩进/标记，便于识别。

### 混合模式（Blend Mode）

用于改变当前图层与下方图层的叠加方式（例如正片叠底、叠加等）。

- 点击混合模式下拉框即可切换。
- 若图层被锁定，混合模式可能无法修改。

## 新建图层

新建图层会在“当前激活图层”上方创建一层。

入口位置可能有两种：

- 默认布局：在图层控制区内有“添加图层”按钮。
- SAI2 布局：常见为图层面板标题区的“添加图层”按钮。

## 图层列表（图层项）

图层列表展示所有图层（通常上方是更“靠上”的图层）。

### 选择与重命名

- 单击图层项：设为当前激活图层。
- 双击图层名称：重命名图层（锁定图层不可重命名）。

### 可见性

每个图层项通常带有“显示/隐藏”按钮：

- 显示：图层参与合成与导出。
- 隐藏：图层不显示（图层项会变暗，预览也会相应变化）。

### 拖拽排序

按住图层项并拖拽，可以调整图层顺序（相当于把图层上移/下移）。

### 常用按钮（不同布局位置略有差异）

在图层项右侧常见操作包括：

- 删除：删除该图层（当只剩 1 个图层或图层被锁定时可能不可用）。
- 锁定/解锁：快速切换锁定状态。
- 剪贴开关：快速切换剪贴蒙版（锁定时不可用）。
- 向下合并：将该图层与下方图层合并（下方图层锁定时不可用）。
- 复制：复制该图层并在上方插入一份副本。

### 文字图层提示

如果图层是“文字图层”，图层项通常会有文字标识。

注意：

- 文字图层上默认不支持大多数像素绘画/选区工具；如需像素级处理，请先栅格化文字图层。

## 图层右键菜单

在图层项上右键（或触控板二指点击）可打开快捷菜单，常见包括：

- 栅格化文字图层（仅文字图层可用）
- 锁定/解锁
- 向下合并
- 创建/释放剪贴蒙版
- 显示/隐藏
- 删除
- 复制

菜单项会根据当前状态自动启用/禁用（例如锁定后无法删除、无法切换剪贴等）。
