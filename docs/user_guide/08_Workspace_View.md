# 08 工作区与视图

本章聚焦“视图相关”的功能：缩放、网格、黑白预览、镜像预览、透视辅助线，以及工作区布局切换。

## 状态信息（顶部信息条）

在画布页面顶部（菜单栏区域）会显示一行状态信息，常见包含：

- 分辨率：画布宽高（像素）。
- 缩放：当前视图缩放百分比。
- 坐标：光标在画布上的坐标（若光标不在画布内则显示 `--`）。
- 网格/黑白/镜像/透视：对应开关的当前状态。

这行信息用于“看状态”，不需要手动操作。

## 缩放（Zoom In/Out）

常见方式：

- 鼠标滚轮 / 触控板滚动：以光标位置为中心缩放。
- 菜单 `View > 放大 / 缩小`：按固定步进缩放。

建议：

- 画细节时适当放大；检查整体构图时缩小。

## 像素网格（Pixel Grid）

用于在高倍率下显示像素网格，便于像素画或检查边缘。

- 开关入口：菜单 `View` 中的“显示/隐藏网格”，或使用快捷键 `Q`。

## 黑白预览（Black/White）

用于用黑白方式预览画面，方便检查明暗关系与对比。

- 开关入口：菜单 `View` 中的“黑白预览”，或使用快捷键 `K`。

## 镜像预览（Mirror）

用于左右镜像预览画面，常用于检查比例、透视和左右不对称问题。

- 开关入口：菜单 `View` 中的“镜像预览”，或使用快捷键 `F`。

## 透视辅助线（Perspective Guide）

透视辅助线用于绘制结构线、建筑线、物体透视等。

### 开关与模式

- 开关入口：菜单 `View` 中的“显示/隐藏透视辅助线”。
- 模式入口：菜单 `View > 透视模式` 中选择“一点 / 两点 / 三点”。

### 调整消失点（基础用法）

开启后，画布上会出现可拖拽的控制点（消失点）。你可以拖动它们来调整透视方向。

提示：

- 使用“拖拽画布（Hand）”工具时，按住 `Shift` 更容易抓取“最近的消失点”并拖动调整。
- “透视画笔（Y）”会依赖透视辅助线工作：未开启透视时无法落笔。

## 工作区布局切换（Workspace Layout）

工作区布局决定面板的摆放方式（功能一致，主要影响操作习惯与空间分配）。

- 入口：菜单 `Workspace > 切换工作区`
  - Default：默认/浮动式布局（工具栏在左侧，右侧为颜色+图层面板）。
  - SAI2：SAI2 风格布局（图层面板与工具面板整合，可拖拽分割）。
- 重置布局：菜单 `Workspace > 重置工作区`，用于恢复默认分割比例与面板尺寸。
