# 02 菜单栏

菜单栏是所有命令入口的总汇。

- Windows / Linux / Web：菜单显示在窗口顶部。
- macOS：菜单显示在系统顶部菜单栏（窗口内可能不显示菜单按钮）。

不同页面/状态下，某些菜单项可能不显示或不可用（例如画布未准备好时无法执行滤镜）。

本章按菜单分组逐项说明。

## File（文件）

用于项目（画布）创建、打开、保存与导出。

- 新建…（Ctrl/⌘+N）：创建一个新项目（新画布）。
- 打开…（Ctrl/⌘+O）：从磁盘打开项目文件。
- 导入图像…（Ctrl/⌘+I）：从文件导入图片（常用于创建画布或作为图层插入，具体行为按对话框流程）。
- 从剪贴板导入图像：从系统剪贴板读取图片并导入。
- 保存（Ctrl/⌘+S）：保存当前项目。
- 另存为…（Ctrl/⌘+Shift+S）：将当前项目保存到新位置/新名称。
- 导出…（Ctrl/⌘+Shift+E）：导出项目（通常用于输出分享或与其他软件互通）。
- 关闭全部（Esc）：关闭所有已打开的项目标签。

## Edit（编辑）

用于撤销/重做，以及剪切、复制、粘贴。

- 撤销（Ctrl/⌘+Z）：回退最近一次操作。
- 重做（Ctrl/⌘+Shift+Z）：恢复刚撤销的操作。
- 剪切（Ctrl/⌘+X）：将选中内容剪切到剪贴板。
- 复制（Ctrl/⌘+C）：将选中内容复制到剪贴板。
- 粘贴（Ctrl/⌘+V）：从剪贴板粘贴内容到画布。

## Image（图像）

用于画布/图像尺寸与整体旋转等操作。

- 图像变换
  - 顺时针旋转 90°
  - 逆时针旋转 90°
  - 顺时针旋转 180°
  - 逆时针旋转 180°
- 图像大小…（Ctrl/⌘+Alt+I）：调整“图像内容”的尺寸（会影响像素内容）。
- 画布大小…（Ctrl/⌘+Alt+C）：调整“画布范围”的尺寸（类似扩展/裁剪画布）。

## Layer（图层）

用于新增图层、合并、栅格化与自由变换。

- 新建
  - 新建图层…（Ctrl/⌘+Shift+N）：在当前图层上方新增一层。
- 向下合并：将当前图层合并到下一层（相当于“合并到下方”）。
- 栅格化：将特定类型图层转为普通像素图层（目前主要用于“文字图层”）。
- 变换（Ctrl/⌘+T）：自由变换当前图层（缩放/旋转/移动等）。

## Selection（选择）

用于快速生成或变换选区。

- 全选（Ctrl/⌘+A）：选中整个画布范围。
- 反选（Ctrl/⌘+Shift+I）：在当前选区基础上反转选择范围。

## Filter（滤镜）

用于画面调整与常用效果。部分功能会以“面板/对话框”的形式出现。

- 边缘柔化…（Ctrl/⌘+Alt+A）：对边缘做柔化/抗锯齿处理（偏向线稿/边缘优化）。
- 收窄线条…（Ctrl/⌘+Alt+N）：让线条变细（常用于清线）。
- 拉伸填色…（Ctrl/⌘+Alt+E）：对填色区域做扩张/延展处理（减少漏白）。
- 高斯模糊…（Ctrl/⌘+Alt+G）：模糊处理。
- 去除漏色…（Ctrl/⌘+Alt+L）：针对填色时的缝隙/漏色进行修正。
- 色相/饱和度…（Ctrl/⌘+U）：调整色相与饱和度。
- 亮度/对比度…（Ctrl/⌘+M）：调整亮度与对比度。
- 色彩范围…（Ctrl/⌘+Alt+Shift+R）：基于颜色范围选择/调整（用于抠色、精细选区等）。
- 黑白…（Ctrl/⌘+Alt+K）：黑白/阈值相关调整（用于线稿提取或预览）。
- 二值化…（Ctrl/⌘+Alt+B）：二值化处理（常用于线稿/掩膜）。
- 扫描纸绘…（Ctrl/⌘+Alt+P）：面向纸上扫描图的清理/增强流程。
- 颜色反转（Ctrl/⌘+Alt+Shift+I）：对颜色做反相处理。

## Tool（工具）

用于调色板、参考图、参考模型等辅助工具。

- 调色板（Palette）
  - 从画布生成调色板…：从当前画面提取颜色生成调色板卡片。
  - 生成渐变调色板：基于当前主色生成渐变调色板卡片。
  - 导入调色板…：从文件导入调色板，并加入菜单中的“已导入调色板”列表。
  - 已导入调色板：选择后会在画布上打开对应调色板卡片。
- 参考图（Reference Image）
  - 创建参考图（Ctrl/⌘+Shift+R）：将当前画布内容导出为图片，并以“参考图卡片”形式浮在工作区上方。
  - 导入参考图…（Ctrl/⌘+Alt+R）：从文件导入图片并打开为参考图卡片。
- 参考模型（Reference Model）
  - Steve 模型：打开内置 Steve 参考模型卡片。
  - Alex 模型：打开内置 Alex 参考模型卡片。
  - 导入模型…：导入 Bedrock 格式（`.json`）模型作为参考模型卡片。

## View（视图）

用于缩放与辅助显示（网格、黑白预览、镜像预览、透视辅助线）。

- 放大（Ctrl/⌘+=）
- 缩小（Ctrl/⌘+-）
- 显示/隐藏网格（Q）：像素网格开关。
- 黑白预览（K）：黑白叠加预览开关。
- 镜像预览（F）：镜像叠加预览开关。
- 透视辅助线
  - 显示/隐藏透视辅助线：透视线开关。
  - 透视模式
    - 一点透视
    - 两点透视
    - 三点透视

## Workspace（工作区）

用于切换工作区布局风格，以及恢复默认布局。

- 切换工作区
  - Default：默认/浮动式布局。
  - SAI2：SAI2 风格布局（面板分割更接近 SAI2 的使用习惯）。
- 重置工作区：将面板分割比例等恢复到默认值。

## Window（窗口）

窗口相关命令（不同平台显示项略有差异），例如最小化、缩放、窗口置前等。
