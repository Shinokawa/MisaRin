# 03 标签栏（项目标签）

标签栏用于管理“打开的项目（画布）”，支持切换、拖拽排序、重命名、关闭，以及新建项目。

## 标签区

每个标签代表一个已打开的项目（画布工作区）。

- 单击标签：切换到该项目。
- 双击标签：进入重命名流程（用于给项目标签改名，便于区分）。
- 未保存提示：标签名右侧会出现一个小圆点，表示该项目有未保存更改。

## “+” 新建按钮

标签栏末尾的 “+” 用于快速创建新项目（等价于菜单 `File > 新建…`）。

## 拖拽排序

按住标签并拖拽，可调整多个项目标签的顺序。

## 关闭与重命名

每个标签右侧有关闭按钮：

- 单击“关闭”：关闭该项目标签（如有未保存内容，可能会弹出提示）。
- `File > 关闭全部（Esc）`：一次性关闭所有已打开项目。

## 文件拖拽（可选技巧）

在支持拖拽导入的系统上：

- 将图片文件拖到标签栏：会为每张图片创建一个新画布项目并打开。
- 将图片文件拖到画布区域：会把图片作为新图层插入到当前项目。

目前支持的拖拽图片格式：`png`、`jpg`、`jpeg`、`bmp`、`gif`。
