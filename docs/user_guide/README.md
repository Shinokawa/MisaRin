# Misa Rin 使用指南

本指南面向日常使用，重点介绍界面中各类“栏位”（菜单栏、标签栏、工具栏、面板）以及每个栏位里可用的工具选项。

- 适用范围：以本仓库当前版本界面为准
- 最后更新：2026-01-07

建议按顺序阅读，也可以从目录直接跳转到你关心的区域。

- 03 标签栏（项目标签）：`03_Title_Tabs.md`
- 04 画布工具栏（左侧工具）：`04_Canvas_Toolbar.md`
- 05 工具设置栏（随工具变化）：`05_Tool_Settings.md`
- 06 颜色栏（颜色/色线/最近颜色）：`06_Color_Panel.md`
- 07 图层栏（图层列表与控制）：`07_Layer_Panel.md`
- 08 工作区与视图：`08_Workspace_View.md`
