# Misa Rin

Misa Rin 是一款聚焦桌面端的数字绘画与像素创作软件，界面基于 Fluent UI，力求在 Windows、macOS、Linux 与 Web 上都提供顺手、简洁且高效的绘画体验。无论是速写、像素草图还是 UI 草稿，都可以在一个统一的画布里完成。

> 项目仍处于早期开发阶段，请勿将其用于生产项目。

## 绘画体验

- **专注画布**：自研位图画布支持任意尺寸画布、无限撤销/重做、实时笔刷反馈与十字准星对齐，便于精准像素操作。
- **多种工具**：画笔、曲线笔、油漆桶、取色器、魔棒、套索/多边形选区、图层移动、拖动画布等常用工具一应俱全，快捷键与提示同步显示。
- **灵活图层**：图层面板支持命名、隐藏、透明度调节与混合模式，为复杂构图提供余地。
- **跨平台一致**：窗口框架、菜单与快捷键在不同平台上保持一致，不需要重新适应。

## 获取与体验

- **桌面版**：无需自行编译，可直接前往 [Releases]([https://github.com/mcdfsteve/misarin/releases](https://github.com/MCDFsteve/MisaRin/releases)) 下载最新安装包或压缩包。
- **网页版**：访问 https://misarin.aimes-soft.com ，在浏览器中即时体验。

## 快速上手

1. 启动应用后根据向导创建新画布或从最近项目列表打开现有文件，即可开始绘制。
2. 如需自行构建，请确保安装 Flutter 3.9+，执行 `flutter pub get` 后运行 `flutter run -d <platform>`（可选）。

> 桌面版默认以最大化形式启动，便于直接投入绘画；macOS 版本会自动挂载系统原生菜单。

## 导入与导出

- **打开项目**：支持从本地文件夹中选择 `.rin` 项目，自动生成缩略图与最近打开记录。
- **导出图像**：通过导出对话框选择倍率与背景色，一键生成 PNG；也可导出透明背景供其它软件继续编辑。
- **项目管理**：内置项目浏览器，可快速定位最近文件、重命名或删除不再需要的项目。
